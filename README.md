# i3_typing_master_2026

A Java-based console application designed to help users assess and improve their typing speed. This project was developed as part of the IPE Midterm Assessment.

## 🚀 Features
- **User Authentication:** Secure Login and Registration screens.
- **Account Management:** Reset password functionality.
- **Typing Test:** Real-time speed assessment using the standard level 1 drills.
- **Progress Tracking:** View a history of your typing test results.
- **Information:** About Us section detailing the development team.
